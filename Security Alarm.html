GSM-based Security Alarm System

What?

  • An automated system that triggers instant messages, calls, and a 12V siren upon unauthorized entry into restricted spaces.
  • Can be deactivated via remote control within 100 m or by SMS.

    • How?

  • Designed circuit board prototype and used CadSoft EAGLE to create schematic, and 2 layer PCB boards.
  • Integrated ultrasonic sensor, PIR sensor, GSM Module-SIM 800L, 12V siren with ATMega328-P.

    • Result

  • Safeguarding over 1000 shops and homes in various cities of Nepal.
