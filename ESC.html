Electronic Speed Controller

What?

  • An Electronic Speed Controller to regulate the speed of an electric motor which is done by varying the amount of power supplied to the motor.
  • Created a complete schematic and designed a four layer (top, GND, GND,bottom) PCB layout.

    • How?

  • Used KiCAD for designing the schematics and the PCB layouts.
  • Integrated essential input protection for all the major components
  • Routed traces minimizing the EMI/EMC issues.
  • Integrated STM32, DRV8032, CAN, USB 3, and MOSFETs.

    • Result

  • A complete PCB layout and associated files for manufacturing a robust ESC that adheres to IPC-7351B and passed all the Design Rule Checker
